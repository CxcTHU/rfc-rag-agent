## 验收后生产默认提升

2026-07-18 的普通 uvicorn 复核确认，统一 `RunCoordinator` 已生效，但 short-loop、route-first 与
retrieval fan-out 仍因配置默认值为 false 而未自动启用。用户确认这些已验收的低延迟策略应成为生产默认：

- `AGENT_SHORT_LOOP_ENABLED=true`
- `PHASE64_ROUTE_FIRST_ENABLED=true`
- `PHASE64_RETRIEVAL_FANOUT_ENABLED=true`

显式 false 只保留为有界诊断/A/B 覆盖，不会恢复第二套 coordinator。两题并发定向复测的后端
`time_to_final_ms` 从 `56240 / 54839` 降到 `26741 / 18573`；该结果是定向回归证据，不是广义
latency release gate。

运行时现在还会在证据充分后、最终模型返回前发出 `final_answer_generating / waiting_final_model`，
React 工作台据此显示“正在等待最终模型返回”，避免界面长期停留在 `evidence_sufficient`。
