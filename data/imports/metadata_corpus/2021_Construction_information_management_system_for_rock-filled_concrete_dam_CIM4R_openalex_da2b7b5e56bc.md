## Abstract

Rock-Filled Concrete (RFC) dam is a new type of concrete dam based on the technology of self-compacting concrete (SCC). To improve the quality control of construction in RFC dam site, a construction information management system for RFC dams (CIM4R) was developed. The software system has two user interfaces. The first is WEB based user interface for PCs, which takes control of the basic project information, the user authorization and data statistical analysis. The second is WeChat based user interface for mobile phones, which is assigned to users for the work stage control, information collection, message notification, and on-site record. Several modules are increasingly developed in the management system, such as inventory and inspection of raw materials, production control in the mixing plant, testing concrete specimens, quality inspection of unit engineering, and so on. AliCloud platform provides the platform for the storage of engineering data.
